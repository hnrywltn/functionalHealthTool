# Changes — April 26, 2026

## Vendors & Pharmacies *(new)*
- New entity type, lives in the sidebar alongside Supplements, Labs, etc.
- Fields: Description, Website (clickable link in view mode), Type, Notes
- Link vendors to any supplement, medication, or other entity via Connections

## Medications
- Added **Prescription (Rx)** field (below Description)
- Added **Cost** field (below Dosage)
- Removed old Vendors / Pharmacies text field — replaced by Vendors & Pharmacies connections

## Diagnoses
- Added **ICD-10 Code(s)** field (multi-entry, appears below Description)

## Supplements
- Added **Indications** field (appears below Description)
- Added **Vendors / Pharmacies** field (multi-entry list, appears below Recommended Brands)
- Added **Documents** field — upload PDFs and images (stored in Cloudflare R2); files open in an inline modal viewer
