# Changes — May 5, 2026

## Site-wide
- Renamed "Bio Markers" → "Biomarkers" throughout the UI (label and plural)
- Textareas in edit mode now auto-expand to fit content — no more scrolling within the field

## New Categories
- Added **Amino Acids** entity (fields: description, role/function, food sources, deficiency signs, normal range, supplemental dosage, notes) — seeded with L-Glutamine and L-Tryptophan
- Added **Genetics** entity (fields: description, gene, variant/SNP, clinical significance, related conditions, notes) — seeded with MTHFR C677T and COMT Val158Met

## Record Duplication
- Added **Duplicate** button on every detail page (sits between Edit and Delete)
- Creates a full copy of the record named "Copy of [original]" and opens it in edit mode

## Hashtag / Tags System
- New `tags` and `entity_tags` tables — any record across all entity types can be tagged
- Tags section in the Connections area of every detail page (always live, no Edit mode required)
- Type `#tagname` or `tagname` and press Enter to add; autocompletes from existing tags
- Clicking a tag navigates to `/tags/[name]` showing all records that share it, grouped by category
- Tags are site-wide — `#MTHFR` on a diagnosis and a supplement will both appear on the `#MTHFR` page

## References
- New **References** section in the sidebar (below all entity categories, separated by a divider)
- Global file library at `/references` — all uploaded files grouped by type (PDFs, Images), each showing linked records
- Upload a file from the References page, name it, and optionally link it to any records
- Each record detail page now has a **References** section (always live, no Edit mode needed)
  - Upload new file directly from a record → inline name prompt → saved to library and linked
  - Or search existing references by label to link a file that's already in the library
- Migrated existing supplement documents into the shared References library
- Clicking a reference label on the References page lets you rename it inline
